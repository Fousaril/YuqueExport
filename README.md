## 用法

- 安装相关依赖


```bash
$ pip3 install pyuque aiohttp huepy PrettyTable

```

- 浏览器访问 [语雀-Token](https://www.yuque.com/settings/tokens) 页面，新建一个读取权限的密钥，然后将该密钥填入`token`变量，最后执行脚本即可

```bash
$ python3 YuqueExport.py

```
